## Learned Workspace Facts

- Site built with Astro 6 (migrated from Next.js): blog posts under `src/content/blog` via Astro content collections; MDX uses remark-math + rehype-katex for maths; Tailwind CSS v4 wired through Vite; deploy via `@astrojs/vercel` with prerendered pages + dynamic Open Graph image route.
- Typography/theme: Geist for body/UI sans, serif (e.g. Instrument Serif) for headings, grey-first palette, deliberate micro-interactions for polished portfolio feel.

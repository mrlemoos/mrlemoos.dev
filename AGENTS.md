## Learned Workspace Facts

- The site is built with Astro 6 (migrated from Next.js): blog posts live under `src/content/blog` via Astro content collections; MDX uses remark-math and rehype-katex for maths; Tailwind CSS v4 is wired through Vite; deployment uses `@astrojs/vercel` with prerendered pages and a dynamic Open Graph image route.
- Typography and theme direction: Geist for body and UI sans, a serif such as Instrument Serif for headings, a grey-first palette, and deliberate micro-interactions for a polished portfolio feel.
